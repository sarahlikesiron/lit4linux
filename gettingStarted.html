What is Linux?

Linux OS is comprised of many different components that piece together the one of a kind OS. These several pieces include:

Bootloader

Software that manages the boot up process of your device. This is usually a screen thaat pops up and goes away after your machine finishes booting up.

Kernel

The kernel is a critical piece of the Linux OS. The Kernel is the core of the system and manages aspects of your machine such as CPU, memory, and peripheral devices like mice and keyboards. It is the lowest level of our OS architecture.

Daemons

This component is in charge of controlling different daemons and how they operate. See daemon defenition next.

Graphical Server

This is a subsystem that displays graphics on the monitor of your device.

Desktop Enviroment

The desktop enviroment is the graphical user interface that the user interacts with when using the operating system. This is the look and feel of the software and can be chosen among many different enviroments.

Applications

Like other operating systems, Linux offers applications available for download and installation on your machine.

Linux's Purpose

Linux is one of the most popular operating systems used for a variety of different reasons. One of the most prevalent reasons for Linux is because of its price. Linux is one of the cheapest options for OS because it is free! Linux costs zero dollars for installation on a machine and can be downloaded and used as much as one desires. In comparison, Linux counterparts including Apple’s MacOS, and Microsoft’s Windows can run up a pretty penny in cost for the software alone. As a result most modern companies will opt for Linux OS on their machines - especially the servers in their organization.


In addition, Linux is open source. Open source essentially means that anybody has the freedom to run it for any purpose. People are free to study the inner workings of Linux and contribute their own changes to it. Furthermore, people are free to distribute copies of your own modified version of it to others. Linux is a community all about freedom and choice.

Installing Centos

CentOS is a type of Linux distribution. A distribution is a type or fork of Linux with different looks and feel. Different distributions are installed for different reasons. If you want your machine to act as a server you might install Ubuntu Server. If you want your machine to act as a personal machine you might install CentOS. The distribution chosen all depends on preference and use case of the user. For beginners we recommend CentOS as its one of the more friendly and intuitive Linux distributions for newer users.
