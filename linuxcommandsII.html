Linux Commands II

Find

The find command allows you to search for files in the given directory. This command is performed hierarchically meaning it starts at the parent directory and moves down to sub-directories within the parent.

A scenario this command might be used is when you want to find all files with the .txt file extension. The command would be “find *.txt”. Note the * in the command is a wildcard meaning that in the space where it is - any text may be there. So, it would find any file regardless of name with the .txt file extension.

Yum

Yum is command used for allowing users to easily install, update, or remove software packages on a machine.

Installing a package with yum: To install an application or software use the command “yum install” followed by the name of the software you wish to install. For example, say you want to download firefox on your machine. Enter the command “yum install firefox”. Watch it install and enter “Y” for yes when prompted if it’s okay to download. Be patient as it can take some time to install depending on the size of the package.

Passwd

The passwd command is a command used in the terminal to change the current password. Simply enter the command “passwd” and you will be prompted to enter the current password and then the new password twice similar to how other systems handle changing passwords.

Wget

Wget is a free utility on CentOS that is used to download files from the web. Enter the command wget followed by the full link for a download of a particular software listed on the link and it will start downloading that software for you. For example, say we want to download a jpeg file off a website. Enter the command “wget” followed by the website name with the file you want.

Ifconfig

Ifconfig is a network utility in Linux to view networking configurations on the machine. Enter the command “ifconfig” and all the network interfaces, ip addresses, MAC addresses, and more will be displayed. This command is great for troubleshooting internet connectivity or other networking issues that might occur on your machine.

Top

Top is a command used to display processes running on the machine. It also displays some valuable metrics regarding your machine including CPU and Memory utilization. Simply enter the command “top” and examine what is going on with your machine.

Su

Su is a command that switches users to the root user of the machine. The root user is the user or account on a Linux machine that by default has access to all commands and files. As you continue your journey learning Linux, you will notice that some commands require the root account to enter it.

Using the command “su” will prompt you to enter the root password. This is the password you set during the installation process of CentOS 7 on your machine. After you enter the password you should now be in root mode with access to everything on the machine. Note: Be careful! Being on root is a security risk as they can do a lot of damage quickly and have access to everything. This is not a mode you want your machine to be in all the time.
