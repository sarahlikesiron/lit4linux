Playing with Text

Text editing is one feature that can also be done completely from the terminal in Linux. CentOS and other Linux distributions offer different types of text editors to use within the terminal to write/edit text, search text, save text, and open files. All of this can be accomplished with different commands in the Linux terminal.

Text Editors:

To get started playing with text you first need to choose a text editor to use. A text editor is a piece of software that allows for the opening of different files and the ability to edit those files. Linux has many different types of text editors available for use but two of the most popular text editors are vim and nano.

Vim is a text editor that is native to nearly every linux distribution by default. Vim is a text editor that is tough to get started with and to master. t can be difficult and confusing starting out. However, most expert Linux users often choose Vim due to the advanced manipulation and power it possesses.

Nano is a text editor that might need to be installed depending on the Linux distribution you are using. In CentOS 7, nano is installed by default. Nano is easy to use and master as well as intuitive for newer Linux users. Nano also lists its shortcut commands at the bottom of the screen when open.

Choosing between these two editors is ultimately up to you. But for the purpose of these tutorials we will be using nano as it is easier to explain as well as getting started with.

Running Nano:

To open the nano text editor and get started working on a file. Enter the command “nano” followed by the filename or full path and filename if the file is not in your current directory. For example, entering the command “nano file1.txt” will open file1.txt in the nano text editor. Note: if the file does not exist, nano will still open anyways and create the file when saved.

Editing within Nano:

Once in the nano text editor screen shown above, you can now edit, add, and make changes to the file that is open. In this example we have the text file file1.txt open in nano. From this point we can easily add words using our keyboard and type and make any changes we want within the file.

Saving a File:

Once you are done editing the file in the nano text editor you probably would like to save your changes to the file so they are not lost. To do this, enter the key combination on your keyboard - “control” + “O” then press enter. This will save your changes to the file open.

Exiting Nano:

After you have saved your changes to the file in nano and you want to exit - use the key combination “control” + “X” to exit out of the nano text editor. You can also see this listed on the bottom of the screen with other shortcuts.
